Exercise 3: C++ Classes

String Class

  • Write a complete StringC as presented in the lectures. Then run the following main program:

int main()
{
    StringC s1("Hello World");
    StringC s2("Goodbye, cruel world");

    s1.Print();
    s2.Print();

    StringC s3=s1+s2;

    s3.Print();
}

  • Put write statements in all the constructors and destructors to see when (and on what string) they are invoked. Can you change the operator+ to eliminate a
    constructor call?
  • Change the class so that it does not use a 0 to terminate the string but instead stores the size internally. Why would you want to do this?
  • Add some more member functions. What about comparison operators like ==, >. What about access operations.
